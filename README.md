This is a sample AI assisted project which is coded mostly by an AI Assistant.

## Tech Stack

- Next.js
- React
- JavaScript
- TypeScript
- Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
